# STRIKE-3
Portable pitch calling device to avoid sign stealing

Components List:  
ESP32-C3-SuperMini (2)  
MAX98357A  
Bone Conduction Speaker  
3.7V 2000mAh Lithium Ion Battery (2)  
TP4056 Battery Charging Board  
10k Ohm Potentiometer
